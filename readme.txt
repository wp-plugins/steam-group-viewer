== Description ==



**(English)**



This plugin allows you to show your steam group profile on your website! It display many features:

- Group name
- Group picture
- Total number of subscribers
- Number of online subscribers
- Number of ingame subscribers
- Number of inchat subscribers
- Number of offline subscribers
- Join button
- 10 user thumbnails'







**(French)**



Ce pluging permet l'affichage des informations importantes d'un groupe steam de votre choix. Il permet l'affichage de nombreux parametres: 

- Nom du groupe
- Photo du groupe
- Nombre d'inscrits
- Nombre d'inscrits en ligne
- Nombre d'inscrits en jeu
- Nombre d'inscrits dans le chat
- Nombre d'inscrits hors ligne
- Boutton "rejoindre"
- 10 miniatures menant aux profiles utilisateurs.






== Installation ==



**(English)**



First of all you need to activate the plugin. After that go to SteamGroup Widget" -> "Settings"  to set your group name and the width of the widget. After that you can add the widget where you want on your 
website by moving it from the "Appearance" -> "Widget" menu.


**(French)**

Tout d'abort, activez le plugin. Rendez vous ensuite sur "SteamGroup Widget" -> "Settings" pour configurer le nom du groupe que vous voulez afficher ainsi que la largeur du widget.
Vous pouvez maintenant ajouter le widget ou vous voulez sur votre site a partir du menu "Apparence" -> "Widget".
